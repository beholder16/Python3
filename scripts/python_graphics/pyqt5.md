# PyQt5

`PyQt5` is not compatible with `PyQt4` 
For example, All Widget stuffs are separated to independent Modules from each modules.
`QtWidgets` is splitted from `QtGui`, like `QtWebKitWidgets` does from `QtWebKit`.
