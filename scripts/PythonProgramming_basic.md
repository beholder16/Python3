# Basics on Python

---
## Write Python Codes

* Comments: ```#```
* Line Continuation: ```\```
* Indent: ```[Tab]``` or ```spaces(2 or 4)```
* Simple Quatation: ```' '``` or ```" "```
* Line-over Quatation(for ```docstring```): ```""" """``` or ```''' '''```
* Assignment: ```=```
